# Tugas-Besar-Struktur-Data

Aplikasi Penerimaan Mahasiswa Baru adalah aplikasi yang berfungsi untuk mendata calon mahasiswa baru dalam melanjutkan pendidikan ke tingkat perguruan tinggi.

Program ini melakukan pendataan dengan menggunakan representasi multi linked list M â€“ N yang memiliki tiga list yaitu list maba, list jurusan, dan list relasi. Ketiga list tersebut menggunakan representasi double circular linked list sebagai pembentuk fungsionalitas yang terdapat dalam program.

Fungsionalitas yang terdapat pada Aplikasi Penerimaan Mahasiswa Baru diantaranya: input, edit, delete, view, searching.
